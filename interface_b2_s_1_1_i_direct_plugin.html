B2S.IDirectPlugin Interface Reference

Interface for plugins for the B2S Server.
All plugins must implement this interface to be recognized by the B2S Server. In addition they also have to export the class implementing the interface using the following attribute [Export(typeof(B2S.IDirectPlugin))] in c# or <Export(GetType(B2S.IDirectPlugin))> for VB.net. Be sure to handle all exceptions of your plugin, since the B2S Server will deactivate plugins throwing unhandled exceptions.
Please refer to the docu on the DirectOutput framework for more info on plugins. More...
